Module 1: Introductie & Installatie

In deze eerste module leer je hoe je een Laravel 12 project opzet, de mappenstructuur begrijpt en je eerste routes maakt.

Een nieuw project starten

Om een nieuw Laravel project te starten gebruik je de composer create-project command of de Laravel installer.

composer create-project laravel/laravel:^12.0 mijn-webshop
cd mijn-webshop
php artisan serve

De mappenstructuur

Laravel heeft een specifieke mappenstructuur. De belangrijkste mappen voor nu zijn:

  • app/: Hier komt al je PHP logica (Models, Controllers).
  • routes/: Hier definieer je de URL's van je applicatie.
  • resources/views/: Hier staan je HTML bestanden (Blade templates).

Je eerste Route

Open het bestand routes/web.php. Hier zie je de standaard route definitie.

use Illuminate\Support\Facades\Route;

Route::get('/', function () {
    return view('welcome');
});

// Een nieuwe test route
Route::get('/hello', function () {
    return 'Hallo Wereld uit Laravel 12!';
});

Parameters in routes

Je kunt ook data meegeven via de URL:

Route::get('/user/{id}', function (string $id) {
    return 'User '.$id;
});
