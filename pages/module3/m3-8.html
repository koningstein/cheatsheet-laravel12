M3.8: Show met Relaties

Leeruitkomsten

  • Ik kan een detail view maken met gerelateerde data (historie).
  • Ik kan conditionele weergave gebruiken (N/A bij null).
  • Ik kan via relaties doorlijsten (bijv. alle prijzen van een product).

Stap 1: Controller

De show methode is simpel dankzij Route Model Binding. We sturen het product naar de view.


public function show(Product $product)
{
    // Omdat we 'prices' relatie hebben in Model, hoeven we niets extra's te doen hier
    return view('admin.products.show', ['product' => $product]);
}

Stap 2: De View (Details)

In de view tonen we de product details én de categorie naam.


<h1>{{ $product->name }}</h1>
<p>Categorie: {{ $product->category->name }}</p>

Stap 3: Relaties Tonen (Lijst)

Omdat een product meerdere prijzen heeft (historie), kunnen we die ook tonen. We loopen door de relatie prices.


<h3>Prijs Historie</h3>
<table>
    <tr><th>Datum</th><th>Prijs</th></tr>

    {{-- Sorteer in de view op datum (nieuwste eerst) --}}
    @foreach($product->prices->sortByDesc('effdate') as $price)
        <tr>
            <td>{{ $price->effdate }}</td>
            <td>€ {{ $price->price }}</td>
        </tr>
    @endforeach
</table>
