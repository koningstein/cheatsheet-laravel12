One to One (HasOne)

Een één-op-één relatie is de simpelste relatie. Een record in tabel A hoort bij precies één record in tabel B.

Voorbeeld: Een User heeft één Profile (met adresgegevens, telefoonnummer, etc.).

1. De 'Eigenaar' (User)

Het model dat niet de foreign key bezit, definieert de relatie als hasOne. Laravel neemt aan dat de foreign key in de andere tabel user_id heet.

Bestand: app/Models/User.php

use Illuminate\Database\Eloquent\Relations\HasOne;

public function profile(): HasOne
{
    return $this->hasOne(Profile::class);
}

2. De 'Gekoppelde' (Profile)

Het model dat de foreign key (user_id) bevat, gebruikt belongsTo. Dit is de inverse (omgekeerde) relatie.

Bestand: app/Models/Profile.php

use Illuminate\Database\Eloquent\Relations\BelongsTo;

public function user(): BelongsTo
{
    return $this->belongsTo(User::class);
}

3. Gebruik

Je kunt de relatie nu benaderen als een eigenschap. Laravel voert de query automatisch uit.


$user = User::find(1);

// Haal het telefoonnummer uit het profiel
$phone = $user->profile->phone;
