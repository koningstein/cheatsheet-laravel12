M2.1: Publieke Layout

De admin-omgeving heeft zijn eigen layout (`layoutadmin`), maar voor de bezoekers van de website willen we een andere uitstraling. We maken hiervoor een publieke layout.

Stap 1: Layout Bestand Maken

Maak een nieuw bestand: resources/views/layouts/layoutpublic.blade.php.

Hierin plaatsen we de HTML-structuur met een navigatiebalk en footer die geschikt is voor bezoekers.


<!DOCTYPE html>
<html lang="nl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mijn Webshop</title>
    <!-- Vite voor CSS/JS -->
    @vite(['resources/css/app.css', 'resources/js/app.js'])
</head>
<body class="bg-gray-100 font-sans leading-normal tracking-normal">

    <!-- Navigatie -->
    <nav class="bg-white shadow p-4">
        <div class="container mx-auto flex justify-between items-center">
            <a href="{{ route('home') }}" class="font-bold text-xl">Webshop</a>
            <div>
                <a href="{{ route('home') }}" class="mr-4">Home</a>
                <a href="{{ route('login') }}">Login</a>
            </div>
        </div>
    </nav>

    <!-- Content -->
    <main class="container mx-auto mt-8 px-4">
        @yield('content')
    </main>

    <footer class="bg-gray-800 text-white text-center p-4 mt-12">
        © 2025 Mijn Webshop
    </footer>

</body>
</html>

Stap 2: Home View & Route

Om de layout te testen, passen we de root-route (/) aan in routes/web.php.


// routes/web.php
Route::get('/', function () {
    return view('home'); // Verwijst naar resources/views/home.blade.php
})->name('home');

Maak vervolgens resources/views/home.blade.php:


@extends('layouts.layoutpublic')

@section('content')
    <h1 class="text-4xl text-center">Welkom in de Webshop</h1>
    <p class="text-center mt-4">Bekijk onze categorieën in het menu.</p>
@endsection
