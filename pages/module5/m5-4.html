M5.4: Orders Tonen

Leeruitkomsten

  • Ik kan een overzicht maken van orders van de ingelogde gebruiker.
  • Ik kan eager loading gebruiken (with('state')).
  • Ik kan een detailpagina maken met de bestelde regels.

Stap 1: Index Methode

We halen alleen de orders op van de huidige gebruiker (Auth::id()). We laden ook de status (state) in.

Bestand: app/Http/Controllers/Open/OrderController.php

public function index(): View
{
    $orders = Order::where('user_id', Auth::id())
                   ->with('state') // Eager load status
                   ->orderByDesc('orderdate')
                   ->get();

    return view('open.orders.index', ['orders' => $orders]);
}

Stap 2: Index View

Maak resources/views/open/orders/index.blade.php.


@extends('layouts.layoutpublic')

@section('content')
    <h1>Mijn Bestellingen</h1>
    <table class="w-full bg-white shadow rounded">

            </tr>
        @endforeach
    </table>
@endsection

Stap 3: Show Methode

Voor de details halen we de order op, inclusief de regels en de producten die daarbij horen.

Bestand: app/Http/Controllers/Open/OrderController.php

public function show(Order $order): View
{
    // Beveiliging: mag deze user deze order wel zien?
    if ($order->user_id !== Auth::id()) {
        abort(403);
    }

    // Laad de regels en de producten
    $order->load('orderrows.product', 'state');

    return view('open.orders.show', ['order' => $order]);
}

Stap 4: Show View

Toon de orderregels in een tabel.


<!-- resources/views/open/orders/show.blade.php -->

        </tr>
    @endforeach
</table>
