Pagination

Als je in je controller Project::paginate(10) gebruikt in plaats van get(), krijg je een speciaal Paginator object terug. Blade maakt het heel eenvoudig om de paginanummers weer te geven.

De Links Weergeven

Het enige wat je in je Blade view hoeft te doen om de knoppen "Volgende" en "Vorige" te tonen, is de links() methode aanroepen op je variabele.

Bestand: resources/views/projects/index.blade.php

@foreach ($projects as $project)
    <div class="project-card">
        {{ $project->title }}
    </div>
@endforeach

<!-- Dit genereert automatisch de HTML voor de paginanummers -->
<div class="mt-6">
    {{ $projects->links() }}
</div>

Tailwind Styling Publiceren

Standaard gebruikt Laravel Tailwind CSS voor de pagination views. Als je deze wilt aanpassen (bijvoorbeeld om ze te vertalen of de kleuren te wijzigen), moet je de views publiceren.


php artisan vendor:publish --tag=laravel-pagination

De bestanden komen nu in resources/views/vendor/pagination/. Je kunt hierin de HTML direct aanpassen.

Handige Methodes

Het paginator object heeft meer handige methodes die je in Blade kunt gebruiken:

  • $projects->count(): Aantal items op de huidige pagina.
  • $projects->total(): Totaal aantal items in de database.
  • $projects->firstItem(): Nummer van het eerste item op deze pagina.
  • $projects->lastItem(): Nummer van het laatste item op deze pagina.

<p class="text-gray-500 text-sm">
    Tonen van {{ $projects->firstItem() }} tot {{ $projects->lastItem() }}
    van de {{ $projects->total() }} resultaten.
</p>
