Tailwind Utility Patterns

Tailwind CSS is geweldig, maar je HTML kan snel veranderen in een "soep" van classes. In Laravel zijn er twee manieren om dit op te lossen: Blade Components en CSS extractie.

1. Blade Components (Aanbevolen)

In plaats van overal dezelfde lange lijst classes te kopiëren voor een knop, maak je een component. Dit is de "Laravel-manier".

Bestand: resources/views/components/button.blade.php

<button {{ $attributes->merge(['class' => 'px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600 transition']) }}>
    {{ $slot }}
</button>

Gebruik:


<x-button class="mt-4">Klik Mij</x-button>
<!-- Resultaat: Heeft alle standaard styles + mt-4 -->

2. CSS Extractie (@apply)

Soms wil je gewoon een ouderwetse CSS class gebruiken, bijvoorbeeld voor een complexe kaart die je in pure HTML gebruikt.

Bestand: resources/css/app.css

@layer components {
    .btn-primary {
        @apply px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600 transition;
    }

    .card {
        @apply bg-white p-6 rounded-lg shadow-md;
    }
}

Gebruik:


<button class="btn-primary">Klik Mij</button>

Advies: Wat te kiezen?

  • Gebruik Blade Components voor UI elementen (knoppen, inputs, alerts). Dit is flexibeler omdat je ook logica (PHP) kunt toevoegen.
  • Gebruik @apply alleen als het echt nodig is, bijvoorbeeld voor styling van Markdown content waar je geen invloed hebt op de HTML-tags.
