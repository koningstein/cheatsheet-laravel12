M1.7: Detailpagina (Show)

Leeruitkomsten

  • Ik begrijp Route Model Binding.
  • Ik kan een detail pagina maken.

Stap 1: Route Model Binding

In plaats van een ID te ontvangen en zelf te zoeken (find($id)), laten we Laravel het werk doen. Door Category $category als parameter te gebruiken, zoekt Laravel automatisch de juiste rij.


public function show(Category $category)
{
    // $category is al gevuld met data!
    return view('admin.categories.show', [
        'category' => $category
    ]);
}

Stap 2: Linken in Index

In index.blade.php maken we een link. We geven het object mee aan de route() helper.


<a href="{{ route('categories.show', $category) }}">Details</a>

Stap 3: Show View

Maak resources/views/admin/categories/show.blade.php en toon de details.


@extends('layouts.layoutadmin')

@section('content')
    <div class="card">
        <h1>{{ $category->name }}</h1>
        <p>ID: {{ $category->id }}</p>
        <p>Gemaakt op: {{ $category->created_at }}</p>

        <a href="{{ route('categories.index') }}">Terug</a>
    </div>
@endsection
