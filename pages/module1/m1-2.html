M1.2: Factory & Seeding

Leeruitkomsten

  • Ik kan een factory aanmaken.
  • Ik kan Faker gebruiken voor testdata.
  • Ik kan een seeder aanmaken en uitvoeren.

1. Factory: De blauwdruk voor data

Om te testen hebben we nepdata nodig. Een Factory definieert hoe één rij eruit ziet.

php artisan make:factory CategoryFactory

In database/factories/CategoryFactory.php gebruiken we Faker:


public function definition(): array
{
    return [
        // Genereer een uniek woord als categorie naam
        'name' => $this->faker->unique()->word(),

        // Optioneel: specifieke datums
        'created_at' => $this->faker->dateTimeThisDecade(),
    ];
}

2. Model koppelen

Zorg dat in je Model (Category.php) de HasFactory trait aan staat:

use Illuminate\Database\Eloquent\Factories\HasFactory;

class Category extends Model {
    use HasFactory;
}

3. Seeder: De database vullen

Een Seeder bepaalt hoeveel rijen er gemaakt worden. 

php artisan make:seeder CategorySeeder

In database/seeders/CategorySeeder.php:


public function run(): void
{
    // Maak 10 categorieën aan
    Category::factory()->times(10)->create();
}

4. Uitvoeren

Vergeet niet de seeder te registreren in DatabaseSeeder.php en draai daarna:

php artisan db:seed
