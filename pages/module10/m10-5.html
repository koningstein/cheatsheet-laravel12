Custom Messages & Vertalingen

Standaard geeft Laravel Engelse foutmeldingen zoals "The title field is required". Dit wil je in een Nederlandse applicatie aanpassen. Dit kan eenvoudig binnen je Form Request.

Berichten Aanpassen

Voeg de messages() methode toe aan je Request class. Hierin map je "veldnaam.regel" naar een tekst.

Bestand: app/Http/Requests/StoreProjectRequest.php

public function messages(): array
{
    return [
        'title.required' => 'Je bent vergeten een titel in te vullen.',
        'title.min'      => 'De titel is wel erg kort, gebruik minimaal :min tekens.',
        'budget.numeric' => 'Het budget moet een geldbedrag zijn.',
    ];
}

Attributen Vertalen

Vaak zijn de standaard zinnen (zoals gedefinieerd in je lang files) prima, maar wil je alleen de veldnaam vertalen. In plaats van "The title field...", wil je "Het Titel veld...". Hiervoor gebruik je attributes().

Bestand: app/Http/Requests/StoreProjectRequest.php

public function attributes(): array
{
    return [
        'title'       => 'projecttitel',
        'description' => 'omschrijving',
        'start_date'  => 'startdatum',
    ];
}

Als nu de validatie faalt, maakt Laravel er automatisch van: "Het projecttitel veld is verplicht." (afhankelijk van je taalbestanden).
