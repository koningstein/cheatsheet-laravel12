M5.5: Order naar PDF

Leeruitkomsten

  • Ik kan de barryvdh/laravel-dompdf package installeren.
  • Ik kan een view renderen naar een PDF bestand.
  • Ik kan een PDF downloaden of streamen naar de browser.

Stap 1: Installatie

composer require barryvdh/laravel-dompdf

Stap 2: Controller Methode

We maken een methode die de PDF genereert. We gebruiken een speciale view (bijvoorbeeld een factuur lay-out).

Bestand: app/Http/Controllers/Open/OrderController.php

use Barryvdh\DomPDF\Facade\Pdf; // Importeer de PDF facade

public function downloadPdf(Order $order)
{
    // 1. Data ophalen
    if ($order->user_id !== Auth::id()) abort(403);
    $order->load('orderrows.product');

    // 2. PDF genereren op basis van een view
    $pdf = Pdf::loadView('open.orders.pdf', ['order' => $order]);

    // 3. Downloaden (of stream() voor bekijken)
    return $pdf->download('factuur-' . $order->id . '.pdf');
}

Stap 3: PDF View

Maak resources/views/open/orders/pdf.blade.php. Let op: PDF's ondersteunen geen complexe CSS zoals Tailwind. Gebruik simpele HTML/CSS (tabellen werken goed).


<html>
<head>
    <style>
        body { font-family: sans-serif; }
        table { width: 100%; border-collapse: collapse; }
        th, td { border: 1px solid #ddd; padding: 8px; }
    </style>
</head>
<body>
    <h1>Factuur #{{ $order->id }}</h1>
    <table>
        <tr><th>Product</th><th>Prijs</th></tr>
        @foreach($order->orderrows as $row)
            <tr>
                <td>{{ $row->product->name }}</td>
                <td>€ {{ $row->price }}</td>
            </tr>
        @endforeach
    </table>
</body>
</html>

Stap 4: Route & Link

Voeg de route toe in web.php:

Route::get('/orders/{order}/pdf', [OrderController::class, 'downloadPdf'])->name('open.orders.pdf');

Plaats een knop op de detailpagina:

<a href="{{ route('open.orders.pdf', $order) }}">Download Factuur</a>
