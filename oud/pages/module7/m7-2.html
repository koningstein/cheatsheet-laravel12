One to Many (HasMany)

Dit is de meest gebruikte relatie. Eén 'ouder' heeft meerdere 'kinderen'.

Voorbeeld: Een Category heeft veel Products. Een Product hoort bij één Category.

1. De 'HasMany' kant (Category)

Gebruik de meervoudsvorm voor de functienaam (products), omdat je een lijst (Collection) terugkrijgt.

Bestand: app/Models/Category.php

use Illuminate\Database\Eloquent\Relations\HasMany;

public function products(): HasMany
{
    return $this->hasMany(Product::class);
}

2. De 'BelongsTo' kant (Product)

Het product bevat de foreign key category_id. We definiëren de relatie terug naar de ouder.

Bestand: app/Models/Product.php

use Illuminate\Database\Eloquent\Relations\BelongsTo;

public function category(): BelongsTo
{
    return $this->belongsTo(Category::class);
}

3. Gebruik


// 1. Van Categorie naar Producten (Lijst)
$category = Category::find(1);

foreach ($category->products as $product) {
    echo $product->name;
}

// 2. Van Product naar Categorie (Enkel)
$product = Product::find(1);
echo $product->category->name;
