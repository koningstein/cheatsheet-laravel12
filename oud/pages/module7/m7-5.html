Query Scopes

Query Scopes zijn herbruikbare stukjes query-logica die je in je Model definieert. Dit houdt je Controllers schoon.

1. Scope Definiëren

Maak een methode die begint met scope, gevolgd door de naam (CamelCase).

Bestand: app/Models/Product.php

use Illuminate\Database\Eloquent\Builder;

// Scope: Active
public function scopeActive(Builder $query): void
{
    $query->where('is_active', true);
}

// Scope: PriceBelow (met parameter)
public function scopePriceBelow(Builder $query, float $amount): void
{
    $query->where('price', '<', $amount);
}

2. Scope Gebruiken

Je roept de scope aan zonder het voorvoegsel scope.

Bestand: app/Http/Controllers/ProductController.php

// Haal alle actieve producten onder de 50 euro
$deals = Product::active()
                ->priceBelow(50)
                ->get();
