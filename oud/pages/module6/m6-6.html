Seeders & DatabaseState

Seeders vullen je database met data. Je roept hierin je factories aan.

1. Factories Aanroepen

In een seeder (bijv. DatabaseSeeder) kun je factories gebruiken.

Bestand: database/seeders/DatabaseSeeder.php

public function run(): void
{
    // Maak 10 gebruikers
    User::factory(10)->create();

    // Maak 5 categorieën, en elke categorie krijgt 10 producten
    // Dit gebruikt de relatie 'products' in het Category model
    Category::factory(5)
        ->has(Product::factory()->count(10))
        ->create();
}

2. Volgorde Bepalen

Als je losse seeders hebt, roep je die hier aan. De volgorde is belangrijk bij foreign keys!


$this->call([
    RoleAndPermissionSeeder::class, // Eerst rollen
    UserSeeder::class,              // Dan users (die rollen nodig hebben)
    ProductSeeder::class,
]);

3. Uitvoeren

Om je database volledig te verversen:

php artisan migrate:fresh --seed
