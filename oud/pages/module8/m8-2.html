Named Routes & Groups

Om je code georganiseerd en onderhoudbaar te houden, kun je routes namen geven en groeperen.

1. Named Routes (Routes een naam geven)

Geef elke route een unieke naam met ->name(). Als je later de URL /user/profile verandert naar /account/me, hoef je je links niet aan te passen, zolang de naam profile.show maar hetzelfde blijft.

Bestand: routes/web.php

Route::get('/user/profile', [UserProfileController::class, 'show'])
    ->name('profile.show');

Gebruik in je code:


// In een Controller (Redirect)
return to_route('profile.show');

// In een Blade View (Link)
<a href="{{ route('profile.show') }}">Mijn Profiel</a>

2. Route Prefixing (URL Voorvoegsels)

Stel je bouwt een admin-paneel. Al je routes beginnen met /admin. In plaats van dit elke keer te typen, gebruik je een groep met een prefix.


// ZONDER groep (herhaling = foutgevoelig)
Route::get('/admin/dashboard', ...);
Route::get('/admin/users', ...);
Route::get('/admin/settings', ...);

// MET groep (schoon en duidelijk)
Route::prefix('admin')->group(function () {

    // De URL wordt automatisch: /admin/dashboard
    Route::get('/dashboard', [DashboardController::class, 'index']);

    // De URL wordt automatisch: /admin/users
    Route::get('/users', [UserController::class, 'index']);

});

3. Name Prefixing (Naam Voorvoegsels)

Net zoals de URL, wil je ook dat de namen van je routes logisch zijn (bijv. admin.dashboard). Je kunt ->name() ook op een groep zetten.

Let op de punt . aan het einde van de naam-prefix!


Route::name('admin.')->group(function () {

    // De volledige naam wordt: admin.dashboard
    Route::get('/dashboard', ...)->name('dashboard');

    // De volledige naam wordt: admin.users.index
    Route::get('/users', ...)->name('users.index');

});

4. De Complete Groep (Alles samen)

In de praktijk combineer je vaak Middleware (beveiliging), Prefix (URL) en Name (route naam).


// Alle routes binnen deze groep:
// 1. Zijn beveiligd (moet ingelogd zijn EN admin zijn)
// 2. Beginnen met /admin
// 3. Hebben namen die beginnen met admin.

Route::middleware(['auth', 'role:admin'])
    ->prefix('admin')
    ->name('admin.')
    ->group(function () {

        // URL: /admin/products
        // Route Naam: admin.products.index
        Route::resource('products', ProductController::class);

        // URL: /admin/orders/archive
        // Route Naam: admin.orders.archive
        Route::get('/orders/archive', [OrderController::class, 'archive'])
            ->name('orders.archive');

    });
