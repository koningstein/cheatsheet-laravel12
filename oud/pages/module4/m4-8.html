M4.8: Winkelwagen Wijzigen

Leeruitkomsten

  • Ik kan het aantal (qty) van een item updaten.
  • Ik kan een item uit de cart verwijderen.
  • Ik weet wat een rowId is en hoe ik die gebruik voor updates.

1. Routes Toevoegen

We hebben routes nodig voor het updaten en verwijderen van items. We gebruiken PATCH voor wijzigingen en DELETE voor verwijderen.


Route::patch('/cart/{rowId}', [CartController::class, 'update'])->name('cart.update');
Route::delete('/cart/{rowId}', [CartController::class, 'destroy'])->name('cart.destroy');

2. Aantal Aanpassen (View)

In de tabel op de index pagina voegen we een klein formulier toe om het aantal te wijzigen. Gebruik $item->rowId.


<!-- Formulier voor update -->
<form action="{{ route('cart.update', $item->rowId) }}" method="POST" class="flex gap-2">
    @csrf
    @method('PATCH') <input type="number" name="qty" value="{{ $item->qty }}" class="w-16 border rounded p-1">
    <button type="submit" class="text-blue-600">Update</button>
</form>

3. Verwijderen (View)

Daarnaast een knop (formulier) om het item volledig te verwijderen.


<form action="{{ route('cart.destroy', $item->rowId) }}" method="POST">
    @csrf
    @method('DELETE')
    <button type="submit" class="text-red-600">Verwijder</button>
</form>

4. Controller Methodes

Voeg de update en destroy methodes toe aan CartController.


public function update(Request $request, $rowId)
{
    Cart::update($rowId, $request->qty); // Update het aantal
    return back()->with('status', 'Aantal aangepast!');
}

public function destroy($rowId)
{
    Cart::remove($rowId); // Verwijder item
    return back()->with('status', 'Item verwijderd!');
}
