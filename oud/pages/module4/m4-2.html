M4.2: De Product Show Pagina

Leeruitkomsten

  • Ik kan een detailpagina maken met Model Binding.
  • Ik kan prijsgeschiedenis tonen met Carbon datum formatting.

Stap 1: Controller Methode

In de ProductController voegen we de show methode toe. Dankzij Laravel's Route Model Binding wordt het product automatisch opgehaald op basis van het ID in de URL.


public function show(Product $product): View
{
    // We sturen het product object direct door naar de view
    return view('open.products.show', ['product' => $product]);
}

Stap 2: Route Toevoegen

Voeg de route toe aan web.php. De {product} parameter zorgt voor de koppeling.


Route::get('/products/{product}', [ProductController::class, 'show'])
    ->name('open.products.show');

Stap 3: View Maken (Details & Historie)

Maak resources/views/open/products/show.blade.php. We tonen hier de productdetails en een lijst met historische prijzen.


@extends('layouts.layoutpublic')

@section('content')
    <div class="container mx-auto py-8">
        <div class="flex flex-col md:flex-row gap-8">
            <!-- Afbeelding -->
            <div class="md:w-1/2">
                <img src="https://placehold.co/600x400" class="w-full rounded-lg shadow-lg">
            </div>

            <!-- Details -->
            <div class="md:w-1/2">
                <h1 class="text-4xl font-bold mb-2">{{ $product->name }}</h1>
                <p class="text-gray-600 text-lg mb-4">Categorie: {{ $product->category->name }}</p>

                <div class="text-3xl font-bold text-green-600 mb-6">
                    € {{ $product->latest_price->price ?? 'N/A' }}
                </div>

                <p class="text-gray-700 mb-6 leading-relaxed">
                    {{ $product->description }}
                </p>

                <!-- Bestelknop komt later in M4.6 -->
                <button class="bg-green-600 text-white px-8 py-3 rounded-lg font-bold hover:bg-green-700 transition">
                    Bestel Nu
                </button>
            </div>
        </div>

        <!-- Prijsgeschiedenis -->
        <div class="mt-12">
            <h3 class="text-2xl font-bold mb-4">Prijsgeschiedenis</h3>
            <table class="w-full bg-white shadow-md rounded overflow-hidden">
                <thead class="bg-gray-100">
                    <tr>
                        <th class="p-3 text-left">Datum</th>
                        <th class="p-3 text-left">Prijs</th>
                    </tr>
                </thead>
                <tbody>
                    @foreach($product->prices->sortByDesc('effdate') as $price)
                        <tr class="border-b">
                            <!-- diffForHumans maakt er "2 months ago" van -->
                            <td class="p-3">{{ \Carbon\Carbon::parse($price->effdate)->diffForHumans() }}</td>
                            <td class="p-3">€ {{ $price->price }}</td>
                        </tr>
                    @endforeach
                </tbody>
            </table>
        </div>
    </div>
@endsection
