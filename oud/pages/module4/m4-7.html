M4.7: Winkelwagen Tonen

Leeruitkomsten

  • Ik kan de inhoud van de winkelwagen ophalen met Cart::content().
  • Ik kan totalen tonen met Cart::total().
  • Ik kan een route en view maken voor het winkelwagen overzicht.

Stap 1: Controller Index

In de CartController maken we de index methode. We sturen de items en het totaal naar de view.


// app/Http/Controllers/Open/CartController.php

public function index()
{
    return view('open.cart.index', [
        'cartItems' => Cart::content(),
        'total' => Cart::total()
    ]);
}

Stap 2: Route Toevoegen

We moeten een URL hebben waar de winkelwagen zichtbaar is. Voeg deze toe aan routes/web.php.


// routes/web.php
use App\Http\Controllers\Open\CartController;

Route::get('/cart', [CartController::class, 'index'])->name('open.cart.index');

Stap 3: View (Tabel)

Maak resources/views/open/cart/index.blade.php. Hierin tonen we de producten in een tabel.


@extends('layouts.layoutpublic')

@section('content')
    <h1 class="text-3xl font-bold mb-6">Winkelwagen</h1>

    @if(Cart::count() > 0)
        <table class="w-full bg-white shadow rounded">
            <thead class="bg-gray-100">
                <tr>
                    <th class="p-3 text-left">Product</th>
                    <th class="p-3 text-left">Prijs</th>
                    <th class="p-3 text-left">Aantal</th>
                    <th class="p-3 text-left">Totaal</th>
                    <th class="p-3"></th>
                </tr>
            </thead>
            <tbody>
                @foreach($cartItems as $item)
                    <tr class="border-b">
                        <td class="p-3">{{ $item->name }}</td>
                        <td class="p-3">€ {{ $item->price }}</td>
                        <td class="p-3">{{ $item->qty }}</td>
                        <td class="p-3">€ {{ $item->subtotal }}</td>
                        <td class="p-3">
                            <!-- Acties komen hier (M4.8) -->
                        </td>
                    </tr>
                @endforeach
            </tbody>
        </table>

        <div class="mt-6 text-right">
            <h3 class="text-2xl font-bold">Totaal: € {{ $total }}</h3>
            <a href="#" class="inline-block mt-4 bg-green-600 text-white px-6 py-3 rounded">
                Afrekenen
            </a>
        </div>
    @else
        <p>Je winkelwagen is nog leeg.</p>
    @endif
@endsection

Stap 4: Menu Teller

Het is handig als je in de navigatiebalk ziet hoeveel items er in je kar zitten. Voeg dit toe aan je layoutpublic.blade.php.


<a href="{{ route('open.cart.index') }}">
    Winkelwagen ({{ \Gloudemans\Shoppingcart\Facades\Cart::count() }})
</a>
