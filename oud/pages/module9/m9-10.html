Wat is Livewire?

Livewire is een full-stack framework voor Laravel dat het bouwen van dynamische interfaces eenvoudig maakt. Normaal gesproken heb je voor interactiviteit (zoals een formulier dat update zonder de pagina te herladen) een JavaScript framework zoals Vue of React nodig.

Waarom gebruiken? Met Livewire schrijf je de logica in PHP (je backend taal) en de template in Blade. Livewire handelt de AJAX-requests, state updates en DOM-manipulatie automatisch af. In Laravel 12 is dit vaak de standaard keuze voor authenticatie-scaffolding.

Een Component Maken

Een Livewire component bestaat uit twee delen: een PHP class (de logica) en een Blade view (de presentatie).

Terminal
php artisan make:livewire Counter

Dit maakt:

  • app/Livewire/Counter.php (De Class)
  • resources/views/livewire/counter.blade.php (De View)

State & Actions (Properties en Methods)

De kracht van Livewire zit in public properties. Deze zijn direct beschikbaar in je view. Als je ze in de view aanpast, veranderen ze ook in de PHP class (Data Binding). Functies in je class noemen we Actions.

app/Livewire/Counter.php

<?php

namespace App\Livewire;

use Livewire\Component;

class Counter extends Component
{
    // Public properties zijn direct zichtbaar in de Blade view
    public $count = 0;

    // Een 'Action' methode
    public function increment()
    {
        $this->count++;
    }

    public function render()
    {
        return view('livewire.counter');
    }
}
resources/views/livewire/counter.blade.php

<div class="p-4 text-center">
    <h2>Huidige telling: {{ $count }}</h2>

    <!-- wire:click zorgt voor de verbinding met de PHP methode -->
    <button wire:click="increment" class="bg-blue-500 text-white px-4 py-2 rounded">
        +1 Optellen
    </button>
</div>

Real-time Data Binding (wire:model)

Voor formulieren gebruik je wire:model. Dit verbindt een input veld direct met een public property in je component. Typ je iets in het inputveld? Dan wordt de PHP-variabele direct bijgewerkt (zonder save knop).

app/Livewire/SearchUser.php

public $search = '';

public function render()
{
    return view('livewire.search-user', [
        // Zoekt live terwijl je typt
        'users' => User::where('name', 'like', '%'.$this->search.'%')->get(),
    ]);
}
resources/views/livewire/search-user.blade.php

<div>
    <!-- .live zorgt voor updates tijdens het typen, anders pas na 'blur' -->
    <input type="text" wire:model.live="search" placeholder="Zoek gebruiker..." class="border p-2">

    <ul>
        @foreach($users as $user)
            <li>{{ $user->name }}</li>
        @endforeach
    </ul>
</div>

Volt (Functionele API)

In moderne Laravel 12 projecten zie je soms ook Volt. Dit laat je de PHP logica binnen het Blade bestand schrijven, vergelijkbaar met de <script setup> van Vue. Dit is optioneel, maar goed om te herkennen.

resources/views/livewire/counter-volt.blade.php

<?php
    use function Livewire\Volt\{state};

    // Definieer state
    state(['count' => 0]);

    // Definieer actie
    $increment = fn () => $this->count++;
?>

<div>
    <h1>{{ $count }}</h1>
    <button wire:click="increment">+</button>
</div>
